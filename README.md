# 📊 Employee Database SQL Analysis

## 📌 Project Overview

This project focuses on analyzing an employee database using SQL.
It demonstrates the use of SQL concepts such as joins, aggregations, window functions, and filtering to extract meaningful insights from structured data.

---

## 🎯 Objectives

* Analyze employee distribution across departments
* Evaluate salary patterns and trends
* Identify top-performing employees
* Understand job roles and organizational structure

---

## 🛠️ Tools & Technologies

* MySQL
* SQL (Structured Query Language)
* MySQL Workbench

---

## 📂 Project Structure

```
employee-sql-project/
│
├── queries.sql              -- Basic SQL queries
├── advanced_queries.sql     -- Window function queries
├── README.md               -- Project documentation
```

---

## 📊 Key SQL Concepts Used

* SELECT, WHERE, ORDER BY
* GROUP BY, HAVING
* Aggregate Functions (COUNT, SUM, AVG, MIN, MAX)
* JOINS (INNER JOIN)
* Window Functions (RANK, ROW_NUMBER, LAG, LEAD)
* Subqueries

---

## 🔍 Key Analysis Performed

### 🔹 Employee Analysis

* Total number of employees
* Employees grouped by gender
* Employees hired after a specific year

### 🔹 Department Analysis

* Number of employees in each department
* Departments with highest workforce
* Department managers

### 🔹 Salary Analysis

* Highest, lowest, and average salary
* Top 5 highest-paid employees
* Salary distribution across departments

### 🔹 Job Title Analysis

* Distribution of job titles
* Common vs rare roles

---

## 📈 Key Insights

* The organization has a large workforce distributed across multiple departments.
* A few departments have significantly higher employee counts, indicating key operational areas.
* Salary distribution shows a wide gap between entry-level and senior roles.
* Only a small percentage of employees earn the highest salaries, reflecting hierarchical structure.
* Certain departments offer higher average salaries, suggesting specialized roles.
* The data reflects a typical corporate pyramid with many employees in lower roles and fewer in leadership positions.

---

## 🚀 Advanced Analysis

Advanced SQL techniques such as window functions were used to:

* Rank employees based on salary
* Identify top earners within departments
* Track salary progression over time
* Compare current and previous salaries
